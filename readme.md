# Persian Spotify - Music Analysis & Recommendation System

Data analysis and music recommendation system for Persian music using machine learning and data mining techniques.

---

## Overview

Comprehensive analysis of Persian music dataset with machine learning-based recommendation engine. Features exploratory data analysis, feature engineering, and predictive modeling to understand listening patterns and generate personalized recommendations.

---

## Technologies & Concepts

**Data Analysis:** Exploratory data analysis (EDA), statistical analysis, data visualization  
**Machine Learning:** Recommendation algorithms, clustering, classification, feature engineering  
**Data Mining:** Pattern discovery, user behavior analysis, music similarity metrics  
**Python Libraries:** pandas, numpy, scikit-learn, matplotlib, seaborn  
**Jupyter Notebook:** Interactive analysis and documentation

---

## Project Structure

